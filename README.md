# Meme Search App

Yeh project React, 
Vite aur Tailwind CSS ka use karke banaya gaya ek simple 
Meme Search application hai. Isme ImgFlip API se memes fetch 
hote hain aur user search bar me type karke memes ko filter kar sakta hai.

## Technologies
- React.js
- Vite
- Tailwind CSS
- React Router DOM
- ImgFlip API

## Features
- Clean gradient UI
- Search bar ke through meme filtering
- API se live meme data fetch
- Responsive meme cards
- Loading skeleton
- Meme details page

## Folder Structure
src/
  components/
  pages/
  App.jsx
  main.jsx
  index.css

## How to Run
npm install  
npm run dev  

Is project me basic React concepts, API fetching aur Tailwind styling ka use kiya gaya hai.
